# 🎉 ThinkFlowAI - Transform Ideas into Structured Knowledge

## 🚀 Overview
ThinkFlow AI 是一个轻量级“从想法到结构化图谱”的工作台。您可以输入一个核心想法，应用会把它扩展为可浏览的知识图谱。这些图谱将帮助您更深入地理解简化复杂的概念。

## 🛠️ Features
- **核心想法输入**: 直接输入您的想法以开始。
- **知识图谱生成**: 自动创建模块和子模块。
- **深入追问**: 每个节点支持继续追问并挖掘更多详细信息。
- **配图生成**: 可选生成与节点相关的配图，提高可视化效果。

## 👨‍💻 System Requirements

## 🎨 How to Use ThinkFlowAI
1. **启动应用**: 在安装完成后，找到ThinkFlowAI图标并双击打开应用。
2. **输入您的想法**: 在主界面输入您的核心想法。
3. **生成知识图谱**: 点击“生成”按钮，应用会创建相应的知识图谱。
4. **探索图谱**: 点击各个节点，使用追问功能深入探索。这将帮助您视觉化信息，并促使更深入的理解。
5. **保存与分享**: 完成后，您可以保存知识图谱或将其分享给他人。
